Advanced

Tip: Store the access token in memory (React state), not localStorage. Use an httpOnly cookie for the refresh token to prevent XSS attacks from stealing your tokens.
Common Mistake: Putting the JWT in localStorage makes it vulnerable to XSS. Also, forgetting to handle token expiry in Axios interceptors leads to silent 401 failures.

Real-time Chat with WebSocket

Advanced

useWebSocket hook with exponential-backoff reconnection, optimistic messages, typing indicator, and read receipts.

Tip: Keep a ref for the reconnect attempt count so it persists across re-renders without triggering them. Debounce the typing indicator to avoid flooding the server.
Common Mistake: Creating a new WebSocket inside useEffect without cleanup causes connection leaks. Always call ws.close() in the cleanup return.

Tip: Store validation rules in a ref, not state. Rules don't change after registration so they don't need to trigger re-renders, and accessing them synchronously in handlers is simpler.
